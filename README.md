# Grim Hollow Player's Guide RPG Companion App
This project contains structured game content from the **Dungeons & Dragons 2024** ruleset, specifically focused on material from the *Grim Hollow Player's Guide* release.
This project is intended as an easily-accessible digital reference for products you already own. Please ensure you only access content in accordance with your local laws.

## Contents

This repository includes data files for:

* **Subclasses** (2/40)
* **Spells**

All material was created within the companion app and published to github to help streamline gameplay and character creation.
